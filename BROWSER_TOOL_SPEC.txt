Tool Name: BrowserAutomationTool

Description/Prompt:
Create a browser automation tool that can:
- Open Brave browser in headless mode
- Navigate to any URL
- Extract all text content from the page
- Find elements by CSS selector or XPath
- Take screenshots and save them
- Close the browser when done

Use self.services.browser for all browser operations.

The tool should have these operations:
1. open_and_navigate - Opens browser and goes to URL, returns page text
2. take_screenshot - Takes screenshot of current page
3. find_element - Finds element by selector and returns its text
4. get_page_content - Returns full page text content (NO parameters needed)

Each operation should handle errors gracefully and return structured results with success status, data, and any error messages.
