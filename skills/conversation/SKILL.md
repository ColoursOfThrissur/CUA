# Conversation Skill

Handle conversations about CUA's capabilities, architecture, and features.

## Purpose
This skill handles conversational interactions about CUA as a solution:
- Questions about CUA's capabilities and tools
- Discussions about CUA's architecture and features
- Explanations of CUA's self-improvement systems
- General conversations about the CUA platform

## When This Skill Activates
- Questions about CUA: "what can you do", "how does CUA work"
- Architecture discussions: "explain your skills system", "how do you improve"
- Feature explanations: "what tools do you have", "how does planning work"
- General CUA conversations without action requirements

## Key Features
- **CUA-Focused**: All responses about CUA as a solution
- **Architecture Aware**: References actual CUA components
- **Solution-Oriented**: Discusses CUA's capabilities and features
- **No Generic AI**: Avoids generic AI assistant responses

## Output Types
- **architecture_explanation**: Details about CUA's systems
- **capability_overview**: List of CUA's tools and features
- **feature_discussion**: Explanation of specific CUA functionality
- **solution_conversation**: General CUA-focused dialogue

## Verification
- **Mode**: none (conversational responses)
- **Risk Level**: low (no system operations)

## Fallback Strategy
If conversation skill fails, ensure responses stay CUA-focused.
