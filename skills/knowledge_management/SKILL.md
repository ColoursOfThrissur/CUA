# Knowledge Management Skill

## Overview
Personal knowledge organization, code snippet storage, note-taking, and information retrieval.

## Capabilities
- **Code Snippet Library**: Store, organize, and retrieve code snippets
- **Note Management**: Create, update, and search personal notes
- **Knowledge Organization**: Tag and categorize information for easy retrieval
- **Search and Discovery**: Find relevant code examples and notes
- **Library Management**: Organize and maintain personal knowledge base

## Preferred Tools
- **LocalCodeSnippetLibraryTool**: Code snippet storage and retrieval
- **LocalRunNoteTool**: Note-taking and management

## Verification Mode
- **output_validation**: Validates successful storage and retrieval operations

## Risk Level
- **Low**: Local storage operations with minimal external dependencies

## Output Types
- **snippet_result**: Code snippet search and retrieval results
- **note_result**: Note creation and retrieval results
- **search_results**: Search query results across knowledge base
- **library_summary**: Overview of stored knowledge and organization

## Use Cases
1. **Code Reuse**: Store and retrieve commonly used code patterns
2. **Documentation**: Maintain personal notes and documentation
3. **Learning**: Organize learning materials and reference information
4. **Project Memory**: Store project-specific notes and code snippets
5. **Knowledge Sharing**: Prepare and organize information for sharing

## Fallback Strategy
Direct tool routing for simple storage and retrieval operations.
